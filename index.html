Challenge 3: Debate

Task: Create visualizations that support or refute a policy debate, with commentary limited to 25 words per chart.

Poverty

Policy question: Have international development efforts successfully reduced extreme poverty?

Global poverty reduction has been a central goal of international development policy for decades. These visualizations examine poverty trends across different income thresholds and world regions to assess progress toward eliminating extreme poverty. The first chart tracks the number of people living in extreme poverty (below $3 per day) by region, while the second shows India's progress across multiple poverty bands, revealing how populations have shifted from extreme poverty toward higher income levels.

Source: World Bank Poverty and Inequality Platform (2025) - with major processing by Our World in Data

Challenge 4: Replication

Task: Find a published chart, replicate it in Vega-Lite, then improve upon the original design.

Green Bond Issuance: China vs Western Nations

Original Chart

This chart was published by the Financial Times, analyzing the global green bond market. The FT's distinctive cream background and clean design makes their charts immediately recognizable. The original visualization effectively shows China's dominance in sustainable finance, but has room for design improvements.

Source: Financial Times | Data: LSEG

My Replication

This replication matches the Financial Times' visual style, including their signature cream background color, grouped bar layout, title positioning, and color palette. The data was traced from the original chart, as the assignment permits approximation when exact data isn't publicly available.

Source: Financial Times (replicated) | Data: LSEG (approximated)

Improved Version

Key improvements made:

  • Changed from grouped bars to line chart for clearer trend visualization
  • Emphasized China with dark green color (thematically appropriate for "green" bonds) and thicker line
  • Western nations shown in muted pastel colors at reduced opacity to provide context without competing for attention
  • Added data points at each year for precision
  • Changed grid lines to dotted for less visual clutter
  • Added country labels at line endpoints to reduce legend dependency
  • Updated title to emphasize the key finding: China's market dominance

Summary (25 words): Emphasized China using thematic green color and transparency. Converted to line chart for trend clarity. Added endpoint labels for better readability.

Challenge 5: Accessing Data - API & Web Scraping

Task (API): Add a chart that uses a live link to an API.

World Bank Development Indicators - Fertility Rate Comparison

This visualization demonstrates real-time data integration using the World Bank API, which provides access to thousands of development indicators across countries and time periods. By connecting directly to the API, the chart automatically updates when new data becomes available, ensuring the visualization remains current without manual intervention.

API Details:

  • Base URL: https://api.worldbank.org/v2/
  • Endpoint structure: country/[country_codes]/indicator/[indicator_code]
  • Format parameter: ?format=json
  • Full URL used: https://api.worldbank.org/v2/country/kor;prk/indicator/SP.DYN.TFRT.IN?format=json&date=1950:2025&per_page=160

The API returns data in JSON format, which Vega-Lite can parse directly to create dynamic, data-driven visualizations. This chart compares fertility rates between South Korea and North Korea over time.

Source: World Bank API - API Documentation
