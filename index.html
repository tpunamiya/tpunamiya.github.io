This portfolio showcases a collection of data visualization projects demonstrating proficiency in economic data analysis, chart design, and interactive visualization techniques. The projects progress from fundamental chart creation to advanced analytical visualizations, utilizing tools including Vega-Lite, Python, and Altair.

Challenge 1: Foundation Charts

Task: Set up GitHub Pages hosting and create initial visualizations using Vega-Lite specifications.

These foundational charts demonstrate the basic capabilities of Vega-Lite for data visualization. The first chart examines economic trends, while the second explores prize money distribution in professional tennis.

Challenge 2: Custom Chart Development

Task: Build custom visualizations using local JSON specifications to display economic indicators.

This visualization presents a custom-built chart analyzing key economic metrics. By creating local specification files, we gain precise control over chart aesthetics and interactivity.

Challenge 3: Inflation Analysis

Task: Analyze Consumer Price Index (CPI) data to understand inflation trends over time.

The Consumer Price Index serves as a crucial indicator of inflation and purchasing power. This line chart tracks CPI movements, revealing periods of high inflation, deflation, and economic stability. Understanding these patterns is essential for economic policy analysis and forecasting.

Challenge 4: Python Integration with Altair

Task: Leverage Python's Altair library to create programmatic visualizations that can be embedded in web portfolios.

Shiller P/E Ratio Analysis

The Shiller Price-to-Earnings (P/E) ratio, also known as the cyclically adjusted price-to-earnings ratio (CAPE), is a valuation metric that smooths out short-term earnings fluctuations. This chart reveals historical market valuations and can signal periods of overvaluation or undervaluation in equity markets.

Python and Altair enable sophisticated data processing and visualization workflows. By generating chart specifications programmatically, we can create reproducible, data-driven visualizations that update dynamically with new data.
