Challenge 5: Accessing Data - API & Web Scraping

Task (API): Add a chart that uses a live link to an API.

This visualization demonstrates real-time data integration using the World Bank API. By connecting directly to the API, the chart automatically updates when new data becomes available, ensuring the visualization remains current without manual intervention.

API Structure:

  • Base URL: https://api.worldbank.org/v2/
  • Endpoint structure: country/[country_codes]/indicator/[indicator_code]
  • Format parameter: ?format=json
  • Full URL used: https://api.worldbank.org/v2/country/kor;prk/indicator/SP.DYN.TFRT.IN?format=json&date=1950:2025&per_page=160

Source: World Bank API - API Documentation

Challenge 8: Big Data

Task: Produce two charts using the UK prices datasets (LRPD or AutoCPI).

Using the Long Run Prices Database (LRPD), I analyzed price differences between men's and women's clothing across three categories: jeans, casual shirts, and jackets. The data was grouped by date and gender, with multiple product variants averaged within each category to create comparable trends from overlapping time periods.

Topic Modeling: Narendra Modi Speech Analysis

Hypothesis: Modi's speeches contain distinct thematic clusters reflecting key policy priorities such as economic development, national security, and social welfare programs.

Using Latent Dirichlet Allocation (LDA), an unsupervised machine learning technique, I analyzed thematic patterns in a collection of speeches delivered by Prime Minister Modi over 2014-18.
